# 🍰 Interactive Recipe Card  

Welcome to the **Interactive Recipe Card** project! 🎉  
This project is designed to provide a **fun, simple, and visually appealing way to cook** with guided steps, ingredients, and cooking modes. Whether you’re a beginner baker 🧑‍🍳 or a seasoned chef 👨‍🍳, this card makes following a recipe smoother than ever.  


---

## ✨ Features  

Here’s what makes this recipe card awesome:  

- 📸 **Recipe Image Display** – A clean, modern layout with an image of your dish.  
- 📝 **Show/Hide Ingredients** – Quickly toggle the ingredients list on and off.  
- 👩‍🍳 **Show/Hide Steps** – Follow step-by-step cooking instructions at your own pace.  
- ⏱️ **Cooking Info** – Displays preparation time and servings.  
- 🍳 **Start Cooking Mode** – A distraction-free view for hands-on cooking.  
- 🖨️ **Print Recipe** – Print out your recipe for offline use.  
- 🔄 **Reset** – Start fresh anytime with the reset option.  

---

## 🎮 How It Works  

1. **Choose a Recipe** → Start with a beautiful card showing the dish image, time, and servings.  
2. **Show Ingredients** → Click the button to view what you’ll need.  
3. **Show Steps** → Step-by-step instructions to guide you.  
4. **Start Cooking Mode** → Go full focus mode while cooking!  
5. **Print Recipe** → Save or print the recipe for offline use.  
6. **Reset** → Done cooking? Reset and try another recipe!  

> It’s as easy as **Click → Cook → Enjoy** 🎂

Cooking is something we all love ❤️ but sometimes recipes feel overwhelming.  
This project was built to:  
- Simplify the **recipe-following process**  
- Provide an **interactive UI** rather than static text  
- Make cooking **fun and beginner-friendly**
